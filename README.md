# vim dotfiles

my vimrc

## about
* ```<C-j> is <esc>```
* ```<Space>a is highlight```
* ```<C-j><C-j> is no highlight```
* ```<Space>i is code format```
* ```<Shift-p><Shift-p> is paste("0p)```
* ```<C-k> is comment out```
